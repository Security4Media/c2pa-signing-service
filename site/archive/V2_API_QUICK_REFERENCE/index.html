API Quick Reference¶

Building & Running¶

# Build
cargo build --package service-http

# Run server
cargo run --package service-http

Endpoint Overview¶

Purpose Async Endpoint Sync Endpoint Input Type
Sign videos POST /v2/c2pa/video POST /v2/sync/c2pa/video File
Sign HLS POST /v2/c2pa/fragmented POST /v2/sync/c2pa/fragmented Folder/HLS
Package media POST /v2/package POST /v2/sync/package Folder
Package + Sign POST /v2/c2pa/package POST /v2/sync/c2pa/package Folder
Job status GET /v2/jobs/{id} - -
Health GET /v2/health - -

Request Template¶

{
  "inputs": [
    {
      "type": "local_file_path|local_folder_path|hls_publication|http_url|s3|gcs|drive",
      "path": "/path/to/input"
    }
  ],
  "params": {
    // Processor-specific parameters
  },
  "output": {
    "type": "local_path",
    "base": "/output/directory"
  },
  "options": {
    "max_files_parallel": 4  // Optional
  }
}

Response Format¶

Async Response¶

{
  "job_id": "uuid",
  "status": "queued|running",
  "status_url": "/v2/jobs/{uuid}"
}

Sync/Status Response¶

{
  "job_id": "uuid",
  "status": "succeeded|failed",
  "results": [
    {
      "input_index": 0,
      "status": "succeeded|failed",
      "output_path": "/path/to/output.mp4",
      "error": null
    }
  ],
  "timing": {
    "queued_at": "2026-03-04T10:00:00Z",
    "started_at": "2026-03-04T10:00:01Z",
    "finished_at": "2026-03-04T10:02:30Z"
  }
}

Common Parameters¶

C2PA Video¶

"params": {
  "media_hint": "video/mp4",        // Optional
  "output_name": "signed_{original}", // Optional
  "options": {
    "timeout_ms": 30000,             // Optional
    "in_place": false                // Optional
  }
}

C2PA Fragmented¶

"params": {
  "output_name": "signed_output",     // Required
  "playlist_pattern": "**/playlist.m3u8", // Optional
  "init_pattern": "**/init_*.mp4",    // Optional
  "frag_pattern": "seg_*.m4s"         // Optional
}

Package¶

"params": {
  "output_name": "packaged_output"    // Required
}

Package and Sign¶

"params": {
  "output_name": "packaged_signed",   // Required
  "playlist_pattern": "**/playlist.m3u8", // Optional
  "init_pattern": "**/init_*.mp4",    // Optional
  "frag_pattern": "*.m4s"             // Optional
}

Input Type Reference¶

// Local file
{"type": "local_file_path", "path": "/path/to/file.mp4"}

// Local folder
{"type": "local_folder_path", "path": "/path/to/folder"}

// HLS publication
{"type": "hls_publication", "path": "/path/to/publication"}

// HTTP URL
{"type": "http_url", "url": "https://example.com/video.mp4", "filename_hint": "video.mp4"}

// AWS S3
{"type": "s3", "bucket": "my-bucket", "key": "path/to/object.mp4", "region": "us-east-1"}

// Google Cloud Storage
{"type": "gcs", "bucket": "my-bucket", "object": "path/to/object.mp4"}

// Google Drive
{"type": "drive", "file_id": "abc123xyz"}
